# Elevator Dashboard - Frontend
This is the frontend repository for the Elevator Dashboard. It is built using ReactJS and provides a user-friendly dashboard to monitor elevator status and details.

## Getting Started
To get started with the frontend, follow these steps:

1. Clone the repository from the provided GitHub link.
2. Make sure you have Node.js installed on your machine.


1. Install the necessary dependencies:
    ```
    yarn install
    ```
1. Start the development server:
    ```
    yarn start
    ```

## Features
* The dashboard displays counts of operational, warning, and out-of-service elevators.
* It also displays all elevator details in tabular form. Table gets refreshed if operational count is clicked and specific information is displayed.
* Clicking on an item (row) in elevator table displays popup with details about it.
